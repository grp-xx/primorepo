# NPL 2026 - Primo Repository

Benvenuti nel repository del corso **Natural Language Processing (NPL) 2026**.

## Descrizione

Questo repository contiene materiale didattico, notebook, esercizi e risorse per il corso NPL 2026.

## Struttura del repository

```
primorepo/
├── notebooks/      # Jupyter notebook per le lezioni
├── data/           # Dataset e file di dati
├── assignments/    # Esercizi e compiti
└── requirements.txt
```

## Requisiti

- Python 3.10+
- [pip](https://pip.pypa.io/en/stable/)

### Installazione delle dipendenze

```bash
pip install -r requirements.txt
```

## Utilizzo

1. Clona il repository:
   ```bash
   git clone https://github.com/grp-xx/primorepo.git
   cd primorepo
   ```
2. Installa le dipendenze:
   ```bash
   pip install -r requirements.txt
   ```
3. Avvia Jupyter:
   ```bash
   jupyter notebook
   ```
